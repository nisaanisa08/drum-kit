Project sederhana untuk mempelajari callback function, event listener, dan DOM manipulation menggunakan tema Drum Kit.
Pengguna bisa menekan tombol drum atau keyboard (w, a, s, d, j, k, l) untuk memainkan berbagai suara drum.

Fitur utama:

    > Event listener untuk click dan keydown
    > Pemanggilan suara menggunakan Audio()
    > Efek animasi tombol dengan class .pressed
    > Studi dasar tentang struktur callback untuk interaksi UI

Cocok sebagai latihan dasar event handling dan manipulasi DOM dalam JavaScript.
